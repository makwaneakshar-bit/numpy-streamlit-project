# Tic Tac Toe Web App 🎮

This is an interactive Tic Tac Toe game. The core game logic and 3x3 grid are built using **NumPy**, and the user interface is created with **Streamlit**.

## Features
* Clean and interactive web UI.
* Automatic Win, Lose, and Draw detection.
* Built entirely in Python.

## How to run this locally:
1. Install the required libraries: `pip install -r requirements.txt`
2. Run the application: `streamlit run app.py`
